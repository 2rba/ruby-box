= ruby-box

Ruby SDK for Box API 2.0 (http://developers.box.com/docs/)

== Contributing to ruby-box
 
* Rename account.example to account.yml and fill in your Box credentials
* Type bundle install
* Type rake.. tests should pass
* Add a failing test
* Make it pass
* Submit a pull request
